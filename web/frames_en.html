They are labeled as follows: time-period_experiment-label - description Choose: 2014_PS-nm-my select[name=s0] Polarstern/2014_PS-nm-my.h5:/raw/PS_mu_nm_data Plot Type Choose a plot type to present the data. There are 4 different types available:
  1. 1D-Histogramm: It creates a frequency diagram of a single variable. The range of values of the variable is given at the x-axis. The ordinate shows the number of entries for a given value-bin.
  2. xy-Plot: It represents the dependency of one variable from another one as it is known from school.
  3. 2D-Histogram: Dependency of one variable from another one (as for xy-Plot). Here, the frequency will be represented by a color scale.
  4. Profile: This plot type will be used mainly in the Detailed Level. A description can be found here.
  5. Map: Plot of the geographical coordinates (longitude and latitude) of the Polarstern research vessel on an Earth map. There is an option to represent an additional variable in dependence on the coordinates as color scale.
